隐私政策

最后更新日期：2026-06-22

WorkBuddy 积分消耗显示扩展（以下简称"本扩展"）尊重用户的隐私权。本隐私政策说明我们如何处理与您相关的数据。

1. 我们收集的数据

本扩展不收集、存储或传输任何个人数据。

2. 我们访问的数据

为了实现功能，本扩展在以下场景中访问数据：

  • 浏览器Cookie：复用 WorkBuddy 网站的登录状态（由浏览器自动管理，扩展不主动读取或存储）
  • API 响应：读取 /billing/meter/get-user-request-usage 接口的响应数据以计算总积分消耗

3. 数据存储

所有数据仅在内存中临时处理，处理完成后即丢弃。扩展不使用 Cookie、LocalStorage 或 IndexedDB 存储任何数据。

4. 数据传输

扩展不会将任何数据传输到第三方服务器。所有数据处理均在用户本地浏览器中完成。

5. 权限说明

本扩展仅请求以下权限：

  • host_permissions：https://www.codebuddy.cn/* — 仅用于在 WorkBuddy 用量页面执行其功能

6. 联系我们

如您对本隐私政策有任何疑问，请访问我们的 GitHub 仓库 提交 Issue。
